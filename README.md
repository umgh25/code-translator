# AI Code Translator

Use AI tools to translate code from one programming language to another.

![AI Code Translator](./public/screenshot.png)

## Running Locally

**1. Clone Repo**

```bash
git clone https://github.com/umgh25/code-translator.git
```

**2. Install Dependencies**

```bash
npm i
```

**3. Run App**

```bash
npm run dev
```
